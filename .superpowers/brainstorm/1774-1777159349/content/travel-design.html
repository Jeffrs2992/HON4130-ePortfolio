How it fits together

  • timeline.json — travel entries added with type: "travel" and highlight: true
  • /travel — filters timeline.json by type === "travel", shows card grid
  • /timeline/:id — existing route, new TravelDetail template handles type "travel"
  • About Me timeline — automatically includes travel entries where highlight: true
  • tagRoutes.js — new tags: travel, study-abroad, road-trip all route to /travel
